#  Tests
Some tasks in part 1 are covered by unit tests.
The unit tests are located in

* [Part1Tests](test/java/com/bmw/manufacturing/part1/Part1Tests.java)

You can run the unit tests
1. Through the "Testing" bar in the sidebar
2. By pressing the green "Play" button when you open the test class.
3. By running `mvn clean test` in the terminal
