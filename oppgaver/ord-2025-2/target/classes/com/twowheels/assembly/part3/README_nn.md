# Del 3 - Klassar og grensesnitt (15%)

I denne oppgåva skal du implementere klassen **CustomBike**, som representerer ein tilpassa sykkel. Denne klassen må implementere grensesnittet **ICustomBuild**.

Relevante stikkord:
* Deklarasjon av klassar, metodar og konstruktørar.
* Implementasjon av grensesnitt.
